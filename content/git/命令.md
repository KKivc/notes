# 工作架构

- 本地工作目录 → 暂存区 (index) → 本地仓库 (commit) → 远程仓库 (push/GitHub)

# 命令

## 初始

- 初始化： git init
- 查看状态：git status
- 添加进暂存区：git add 文件

- 将暂存区文件提交本地库：git commit -m "信息" 文件

- 查看日志：git log
- 版本穿梭：git reset --hard 版本号

## 分支

- 查看分支：git branch -v

- 创建分支：git branch 分支名

- 切换分支：git checkout 分支名

- 合并分支：git merge 分支名

## 连接远程库

- 查看远程库链接别名：git remote -v
- 添加链接别名：git remote add 名字 链接
- 重命名当前分支：git branch -M main
- 第一次先关联链接名和分支，后续可以不用写：git push -u 别名 分支
- 推送：git push 别名 分支名

- 拉取：git pull 别名 分支名
- 克隆：git clone 链接
