## 项目简介

NetHub 是一个基于 **Hugo** 静态网站生成器和 **Hextra** 主题构建的个人知识库网站。

### 主要功能

- 📝 **Markdown 驱动** — 所有内容以 Markdown 格式管理
- 🔄 **飞书自动同步** — 通过飞书文档自动同步到 GitHub
- 🚀 **Docker 部署** — 容器化部署，自动更新
- 📱 **响应式设计** — 桌面端和移动端完美适配

### 技术栈

| 组件 | 技术 |
|------|------|
| 框架 | Hugo |
| 主题 | Hextra |
| 部署 | Docker / Nginx |
| 同步 | 飞书 → GitHub Webhook |
