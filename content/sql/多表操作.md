#  交叉查询

- 遍历，职工表的每一个员工和每一个部门进行匹配，都会显示出来，并没有筛选。

```sql
select * from dep,emp;
```

# 内连接查询

- 交叉查询的基础上添加条件，求交集

```sql
select * from dep,emp where dept_no = dept_id;
#or
select * from dep join emp on dept_no = dept_id;
```

# 外连接查询

- 左外连接、右外连接：以左/右表为主，另一张表对比输出，没有的就是null，可以相互换表的顺序
- 满外连接：两张表的并集，上下拼接，左外连接和右外连接合并并去重
- on后面为条件，多条件用and/or连接

```sql
#语法
select * from A
	left outer join B on 条件1
	left outer join C on 条件2
	left outer join D on 条件3；

#左外连接  左表为主
select * from dep left outer join emp on dept_no = dept_id;

#右外连接  右表为主
select * from dep right outer join emp on dept_no = dept_id;

#满外连接
select * from dep left outer join emp on dept_no = dept_id
union
select * from dep right outer join emp on dept_no = dept_id
```

# 子查询

- 单行单列
- 单行多列
- 多行单列
- 多行多列

```sql
#查询最大年龄  先查年龄再带入
select * from emp where age = (select max(age) from emp);

#查询研发部和销售部的员工信息		两个部门=两个部门号，所以不能用=，用in
select * from emp where dept_id in (select deptno from dep where name= '研发部' or name = '销售部');
```

## 关键字

### all

- 与子查询返回的所有值进行比较为true，返回true
- 结合=、>、<、>=、<=、!=使用

```sql
#查询年龄大于1003员工的员工信息
select * from emp where age > all(select age from emp where dept_id='1003');
```

### any/some

- 与子查询返回的任何值进行比较为true，返回true，只要有一个满足则为true
- 结合=、>、<、>=、<=、!=使用
- any和some用法一样

```sql
#查询年龄大于1003任意员工的信息
select * from emp where age > any(select age from emp where dept_id='1003');
```

### in

- 用于判断某个记录的值，是否再指定集合里
- 可以前面加not将条件反过来

```sql
#查询研发部和销售部的员工信息包含员工号和员工名字
select id,name from emp where dept_id in(select dept_no from dept where name = '研发部' or name = '销售部'); 
```

### exists

- 先查询子查询的内容，如果有数据结果，则exists()结果为true，外层查询执行
- 如果没有数据，则exists()结果为false，外层不执行
- 大数据量的时候使用
- 需要给每张表起别名

```sql
#查询有所属部门的员工信息
select * from emp a where exists(select * from dept b where a.dept_id = b.dept_no);
```
