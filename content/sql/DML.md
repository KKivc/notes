> 对数据进行增删改

# 操作

- 在表中添加数据 insert
- 修改数据 update
- 删除数据 delete

## 数据插入

- 添加行

### 给指定列添加数据

`insert into 表名(列1，列2，...) values(值1,值2，...);` #列与值一一对应

### 给全部列添加数据

`insert into 表名 values(值1,值2,...);`

### 批量添加数据

```sql
insert into 表名(字段1，字段2，...) values(值1，值2，...),(值1，值2，...),(值1，值2，...);
insert into 表名 values(值1，值2，...),(值1，值2，...),(值1，值2，...);
```

```sql
insert into stu(id,name,gender,age,birth,address,score)
            VALUES(1,'aa','男',18,'2020-01-01','北京',86.5),
                   (2,'bb','男',15,'2020-01-11','北京',87.6),
                   (3,'cc','女',18,'2020-01-16','北京',90.5),
                   (4,'dd','男',18,'2020-01-25','北京',86.5);

```



### 注意

- 插入数据时，指定字段顺序需要与值的顺序一一对应

- ==字符串==和==日期型数据==要包含在==单引号==中
- 插入的二数据大小，在字段的规定范围内

## 修改

`update 表名 set 字段名1=值1, 字段2=值2, ...[where 条件]; `    //在...条件下，把字段1的值改为值

- 修改语句的条件可以有，也可没有，如果没有，会修改整张表的==所有数据==

## 删除

- delete删除表后，重新插入数据，自增长的值还是从原来最后的值加1
- truncate则自增长也清空，从1开始

`delete from 表名 [where 条件];`

### 清空表数据

`truncate table 表名;`

### 注意

- 删除语句的条件可以有，也可没有，如果没有，会删除整张表的所有数据
- 不能删除某一字段的值（可以使用update）
