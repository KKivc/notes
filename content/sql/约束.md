---
title: 约束
---

- 在建表的时候的操作
- 对表中数据的约束条件
- eg：用户表的有些列的值(手机号)不能为空

# 主键约束

- 是一个列/多个列的组合，能唯一标识表中的每一行
- 主键约束的列==不允许重复==、==不允许出现空值==
- 每个表只允许一个主键
- 关键字：primary key

## 添加单列主键

- 定义字段的同时定义主键

```sql
#语法
CREATE TABLE if not EXISTS db1.emp(
  id int PRIMARY KEY, #主键约束，id具有唯一性
  name VARCHAR(10),
  gender VARCHAR(1),
  salary FLOAT
);

```

- 定义完字段之后指定主键

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int ,
  name VARCHAR(10),
  gender VARCHAR(1),
  salary FLOAT,
  PRIMARY key(id)
);

```

## 添加联合主键

- 当主键由多个字段组成时，只能定义完字段后再统一指定主键
- 联合主键也是一个主键
- 如果是联合主键，那么只要值不是完全一样就可以，但是不能为NULL

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int ,
  name VARCHAR(10),
  gender VARCHAR(1),
  salary FLOAT,
  PRIMARY key(id,name)
);

```

### 修改表结构添加主键

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int ,
  name VARCHAR(10),
  gender VARCHAR(1),
  salary FLOAT,
);

alter table emp2 add primary key (id,name);

```

## 删除主键

- 不分单列主键还是联合主键

```sql
alter table emp2 drop primary key;
```

# 自增长约束

- auto_increment

- 当定义为自增长后，该值就不需要输入数据了，由数据库系统根据定义自动赋值，每增加一条记录，会自动以相同的步长增长
- 默认每新增一条记录，字段值自动加1
- 一个表中只有一个字段使用auto_increment约束，且为单列主键
- 约束的字段必须具备NOT NULL的属性
- 约束的字段只能是整数类型
- 最大值受该字段的数据类型约束，达到上限则失效

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int primary key auto_increment,
  name VARCHAR(10),
  gender VARCHAR(1),
  salary FLOAT,
);

```

## 指定自增字段初始值

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int PRIMARY key auto_increment,
  name VARCHAR(10),
)auto_increment=100;	#id从100开始，101，102

```

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int PRIMARY key auto_increment,
  name VARCHAR(10),
);
#建表完成后指定
alter table emp2 auto_increment=100;
```

# 非空约束

- 字段值不能为空 not null
- 可以设置多个

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int ,
  name VARCHAR(10) not null,
  gender VARCHAR(1) not null,
);

#创建完表之后定义
alter table emp2 modify name VARCHAR(10) not null;

#删除非空约束
alter table emp2 modify name VARCHAR(10);
```

# 唯一约束

- 字段的值不能重复，可以为NULL

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int unique,
  name VARCHAR(10) ,
  gender VARCHAR(1),
);


#创建完表之后定义
alter table emp2 unique(id);

#删除
alter table emp2 drop index id;
```

# 默认约束

- 用来指定某列的默认值

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int,
  name VARCHAR(10) ,
  gender VARCHAR(1) default '男',
);

#创建完表之后定义
alter table emp2 modify gender varchar(1) default '男';

#删除默认约束
alter table emp2 modify gender varchar(1) default null;
```

# 零填充约束

- 插入数据时，当字段的值长度小于定义的长度，会在该值前面补0
- zerofill默认为int(10)

```sql
CREATE TABLE if not EXISTS db1.emp2(
  id int zerofill,
  name VARCHAR(10) ,
  gender VARCHAR(1),
);

#删除
alter table emp2 modify id int;
```

# 外键约束

- foreign key
- 一个主表，一个从表

```sql
#主表
CREATE TABLE if not EXISTS db1.dept(
  dept_no int primary key,  --部门号 主键列
  name VARCHAR(10) ,
  gender VARCHAR(1),
);

#从表
CREATE TABLE if not EXISTS db1.emp2(
  id int primary key,
  name VARCHAR(10) ,
  gender VARCHAR(1),
  det_id int，  --员工所属部门
  constraint emp_fk foreign key(dept_id) references dept(dept_no)
);

```
