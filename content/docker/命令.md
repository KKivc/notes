命令
使用国内阿里云镜像源加速器
系统
systemctl start/stop/restart/status docker：启动/停止/重启/状态 docker
systemctl enable/disable docker：开机自启/关闭
docker info：查看docker概要信息
docker --help：帮助文档
docker 具体命令 --help：命令帮助
镜像
查看
docker images：查看本地主机镜像
-a：本地所有镜像
-q：只显示镜像id
docker search  [选项] 镜像名字：查找镜像
--limit 5：列出5个
docker system df ：查看镜像/容器/数据卷所占空间
安装
docker pull 镜像名字[:版本号] ：没有版本号默认最新版=latest
删除
docker rmi  [选项] 镜像名/id：删除镜像
-f：强制删除
docker rmi -f $(docker images -qa)：删除本地所有镜像
虚悬镜像
构建时或删除镜像时出现了错误
仓库名、tag都是none的镜像
docker image prune：删除所有的虚悬
容器
一般-d后台启动程序，exec进入对应容器
创建容器
docker run [选项] 镜像名 [/bin/bash]
选项
--name=容器新名字：为容器指定一个名称
-d：后台运行容器并返回容器id，启动守护式容器
-i：交互式模式运行容器，通常与-t配合使用
-t：为容器重新分配一个伪输入终端，通常与-i配合使用
-P:随机端口映射
-p：指定端口映射
--cluster-enabled yes：开启集群
--appendonly yes：持久化
--privileged=true：开启权限
docker ps [options] ：列出当前所有正在运行的容器
options：
-a：当前+历史运行过的容器
-l：显示最近创建的容器
-n 数字：显示最近n个创建的容器
-q：静默模式，只显示id
退出
exit：退出后，容器停止
ctrl+p+q：退出，容器不停止
开启/停止/重启/强制停止
docker start/stop/restart/kill id/容器名字
删除
docker rm id/容器名：删除容器
-f：强制删除
查看容器日志
docker logs id/容器名
以交互式进入在后台运行的容器
docker exec -it 容器名 /bin/bash ：exit退出不会停止进程
docker exec -it id redis-cli
docker attach id/容器名 ：exit退出停止进程
容器文件—>主机
docker cp id/容器名:容器内路径 目的主机路径
导入/导出容器备份
docker export id/容器名 > 文件名.tar
cat 文件.tar | docker import - 镜像用户/镜像名:镜像版本号
commit容器副本成为新镜像
docker commit -m="提交的描述信息" -a="作者" 容器id 创建的目标镜像名[:tag]
阿里云镜像本地推拉
容器数据卷
宿主机的文件夹挂载到docker容器内
默认读写
docker run -it --privileged=true -v 宿主机绝对路径目录:容器内目录 镜像名 
docker inspect 容器id：查看挂载的目录
docker run -it --privileged=true -v 宿主机绝对路径目录:容器内目录:ro 镜像名 ：只读
docker run -it --privileged=true --volumes-from u1 宿主机绝对路径目录:容器内目录  --name u2 镜像名 ：宿主机同个目录挂载两个同样的镜像，一个父类一个子类，三台/*文件共享
