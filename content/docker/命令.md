# 系统

- 使用国内阿里云镜像源加速器

- ```bash
  #启动/停止/重启/状态 docker
  systemctl start/stop/restart/status docker
  
  #开机自启/关闭
  systemctl enable/disable docker
  
  #查看docker概要信息
  docker info
  
  #帮助文档
  docker --help
  
  #命令帮助
  docker [具体命令] --help
  ```

# 镜像

## 查看

```bash
#查看本地主机镜像
docker images [option]
				-a：本地所有镜像
				-q：只显示镜像id
				
#查找镜像
docker search  [选项] [镜像名字]
				--limit 5：列出5个
				
#查看镜像/容器/数据卷所占空间
docker system df 
```

## 安装

```bash
#没有版本号默认最新版=latest
docker pull 镜像名字[:版本号] 
```

## 删除

```bash
#删除镜像
docker rmi  [选项] [镜像名/id]
			-f：强制删除
			
#删除本地所有镜像
docker rmi -f $(docker images -qa)
```

## 虚悬镜像

- 构建时或删除镜像时出现了错误

- 仓库名、tag都是none的镜像

- ```bash
  #删除所有的虚悬
  docker image prune
  ```

# 容器

- 一般-d后台启动程序，exec进入对应容器

## 创建容器

```bash
#运行容器
docker run [选项] [镜像名] [/bin/bash]
			--name=容器新名字：为容器指定一个名称

			-d：后台运行容器并返回容器id，启动守护式容器

			-i：交互式模式运行容器，通常与-t配合使用

			-t：为容器重新分配一个伪输入终端，通常与-i配合使用

			-P:随机端口映射

			-p：指定端口映射

			--cluster-enabled yes：开启集群

			--appendonly yes：持久化

			--privileged=true：开启权限
			

#列出当前所有正在运行的容器
docker ps [options] 
			-a：当前+历史运行过的容器
			-l：显示最近创建的容器
			-n 数字：显示最近n个创建的容器
			-q：静默模式，只显示id
			

#退出容器
exit：退出后，容器停止
ctrl+p+q：退出，容器不停止

#开启/停止/重启/强制停止 容器
docker start/stop/restart/kill [id/容器名字]

#删除容器
docker rm [id/容器名]
		 -f：强制删除

#查看容器日志
docker logs [id/容器名]

#以交互式进入在后台运行的容器
docker exec -it [容器名] /bin/bash   #exit不会退出停止进程
docker attach [id/容器名]            #exit退出停止进程

#容器文件—>主机
docker cp [id/容器名:容器内路径] [目的主机路径]

#导入/导出容器备份
docker export [id/容器名] > [文件名.tar]
cat [文件.tar] | docker import - [镜像用户/镜像名:镜像版本号]
#commit容器副本成为新镜像
docker commit -m="提交的描述信息" -a="作者" [容器id] [创建的目标镜像名:tag]


```

## 容器数据卷

- 宿主机的文件夹挂载到docker容器内

- ```bash
  #默认读写
  docker run -it --privileged=true -v [宿主机绝对路径目录:容器内目录] [镜像名]
  
  #查看挂载的目录
  docker inspect [容器id]
  
  #只读
  docker run -it --privileged=true --volumes-from u1 [宿主机绝对路径目录:容器内目录]  --name u2 [镜像名] 													#宿主机同个目录挂载两个同样的镜像，一个父类一个子类，三台文件共享
  
  
  ```
