IP
ip a  查看网络接口 IP 地址配置信息
ip a show eth0  只显示 eth0 的信息
ip -4 a  只显示 IPv4 地址
ip -6 a  只显示 IPv6 地址
ip link  只显示接口状态不显示ip
dhcp
启用网卡 → 获取 IP
sudo ip link set eth0 up
sudo dhclient eth0
ip a查看输出中是否有 dynamic 关键字，有就说明使用了dhcp
route
ip route  查看路由
default via 192.168.1.1 dev eth0 proto dhcp metric 100 192.168.1.0/24 dev eth0 proto kernel scope link src 192.168.1.100 metric 100 
default via 192.168.1.1 dev eth0  默认网关
192.168.1.0/24 dev eth0   192.168.1.x直连网络，直接从eth0发，不需要网关
src 192.168.1.100  本机使用的源ip
proto dhcp  这条路由是由 DHCP 自动添加的
metric 100  路由优先级（数字越小优先级越高）
ip route show table all    显示所有路由表（包括多表）
ip route get 8.8.8.8       查看访问某个 IP 会走哪条路由
添加路由
sudo ip route add default via 192.168.1.1  添加默认网关
sudo ip route add 10.0.0.0/24 via 192.168.1.100    添加静态路由访问 10.0.0.0/24 走 192.168.1.100
删除路由
sudo ip route del 192.168.1.0/24
接口
lo - 本地回环接口（127.0.0.1）
eth0/enp2s0 - 有线网卡，只能靠物理插拔
wlan0/wlp - 无线网卡，可以用指令控制
state UP/DOWN - 接口是否启用
inet - IPv4 地址
inet6 - IPv6 地址
mtu - 最大传输单元
curl
curl [选项] URL
选项
-s：静默输出
-I：只获取响应头
-o file：保存输出到文件
-v：显示详细过程
-L：跟随重定向
-H;自定义header
-d：post发送数据
-X：指定请求方法
