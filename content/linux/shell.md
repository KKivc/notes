shell
脚本以 #!/bin/bash 开头
脚本需要有可执行权限 | sh xxx.sh不需要权限直接执行
...||...前面执行失败，执行后面
...&&...前面成功才执行后面
${}：变量扩展，对变量/参数操作
$()：执行命令/定义数组
[]：条件测试，判断真假
$(()):运算符内不用写“”、$，直接写变量名，$(( SUM+A ))计算并返回结果
(()):前面不加$,计算不返回结果
[[]]:通配符匹配，[[ "$file" == *.log]]判断后缀名/字符串
引用文件$"file"
引用后面是字母的话不用“”，用{}，{$SUM}KB
变量
分为系统变量和自定义变量
系统变量：$HOME、$PWD、$SHELL...
显示当前shell所有变量：set
自定义变量
定义
定义变量：变量名=值    //不要有空格
撤销变量：unset 变量
静态变量：readonly 变量名=值，不能反复的被初始化，只定义一次，不能unset
输出变量需要加$
环境变量
在/etc/profile配置环境变量
定义环境变量：export 变量名=值，将shell变量输出为环境变量/全局变量
source 配置文件，让修改的配置信息立即生效
位置参数
通过预留位置，在终端传递参数
$n：n为数字，$0代表命令本身，$1-$9代表第一到第九个参数，十个以上的参数需要用大括号包含，${10}
$*：代表命令行中所有的参数，把所有参数看成一个整体
$@:代表命令行中所有参数，把每个参数区分对待
$#:代表命令行中所有参数的个数
变量=${参数:-默认值}
${1:-.}  第一个参数，有参走参，没参走当前目录
${3:-8080}  第三个参数，默认 8080 端口
参数展开
#：删除最短匹配前缀
##：删除最长匹配前缀
%：删除最短匹配后缀
%%：删除最长匹配后缀
##*.  删除前缀
%%.*删除后缀
模式
.  匹配一个点本身
*  匹配任意长度任意字符
*.  前面任意字符+一个点
.*  一个点+后面的任意字符
"${file##*.}"
预定义
预先定义好的变量，可以直接在shell中使用
$$:当前进程的PID
$!:后台运行的最后一个进程的PID
$?:最后一次执行的命令返回的状态，0：上一个命令正确执行；非0：上一个命令不正确
退出码
exit 0：成功
1：一般性错误
2：误用shell内置命令
126：命令不可执行
127：命令未找到
130：被ctrol c终止
规则
变量名称可以由字母、数字、下划线组成，不能以数字开头。
等号两边不能由空格
变量名称一般大写
将命令的返回值赋给变量
A=‘date’  反引号，不是单引号，运行里面的命令，并把结果返回给变量A
A=$(date)等价于反引号
单行注释：#  ；多行注释： :<<!内容！
运算符
$((运算式)) |  $[运算式] | expr m + n
expr运算符之间要有空格
expr \* , / , % 乘，除，取余
条件判断
[ condition ] condition前后要有空格，非空返回true ，可用$?验证(0为true，>1为false)
= 字符串比较
两个整数比较
-lt 小于
-le 小于等于
-eq 等于
-gt 大于
-ge 大于等于
-ne 不等于
按照文件权限进行判断
-r 有读权限
-w 有写权限
-x 有执行权限
按照文件类型进行判断
-f 文件存在并是一个常规文件
-e 文件存在
-d 文件存在并是一个目录
-z：字符串为空
-n：字符串不为空
流程控制
if
if和[之间有空格，条件两边也有
单分支

if [ condition ]
then
    代码
fi
多分支

if [ condition ]
then 
    代码
elif [ condition ]
then
    代码
fi
case

case $变量名 in
"值1")
#如果变量值等于值1，则执行程序1
程序
;;
"值2")
程序
;;
...
*)
#如果都不是以上的值，则执行此程序
程序
;;
esac 
for
continue：跳过当前
break：跳过后续所有

for 变量 in 值1 值2 值3...
do
    程序
done

for ((初始值;循环控制条件;变量变化))
do
    程序
done
while
while和[之间有空格，条件两边也有

while [ condition ]
do
    程序
done
read
read读取控制台输入
read 选项 变量名
选项：
-p：指定读取值时的提示符
-t：指定读取值时等待的时间（s），如果没有在指定时间输入，就不再等待
函数
系统函数
basename
返回完整路径最后/部分，常用于获取文件名
basename [pathname] [后缀名]
basename /home/aaa/test.txt ——>text.txt
basename /home/aaa/test.txt .txt ——>text
dirname
返回完整路径，用于返回路径部分
dirname 文件绝对路径
dirname /home/aaa/text.txt  ——>/home/aaa
自定义函数

function funname[()]
{
    程序
    [return int;]
}

#调用直接写函数名 funname [值]
数组
普通数组
数字索引
arr=(a b c)
echo "${(arr[0])}"     要加{}
关联数组
declare -A map      ：需要定义
map=(["key1"]="val1" ["key2"]="val2")
echo "${map[key1]}"
