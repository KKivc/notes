命令
df -h  查看磁盘使用情况
lsusb  显示所有连接到计算机的 USB 设备
sudo apt update  更新软件源
sudo apt install  装软件
curl -v http://www.baidu.com  测试网络连接和 HTTP 请求详细信息
 curl -I http://www.baidu.com  只获取 HTTP 响应头信息的命令，不下载网页正文内容
-a：列出所有文件包括以.开头的隐藏文件
-l：单列输出，含有权限的输出
关机
halt 关机
reboot 重启
sync 把内存同步到磁盘
shutdown -h now 立刻关机
shutdown -h 1      1min后关机
shutdown -r now    重启
用户管理
普通用户->管理员：su - root
增加普通用户： useradd 用户名        添加密码：passwd 用户名
删除普通用户：userdel 用户名  
删除普通用户及主目录：userdel -r 用户名 
id 用户名：查询用户信息

帮助指令
man 命令/配置文件
help 命令
文件目录指令
绝对路径：从根目录/开始定位
相对路径：从当前位置开始定位
pwd：显示当前用户所在目录的绝对路径
创建
mkdir /home/tt ：mkdir只能创建一个目录
mkdir -p /home/tt/y ：创建多级目录
touch 文件：创建空文件
删除
rm 文件：只能删除空文件夹、文件
rm -rf 文件：删除非空目录    -r：整个文件夹    -f：强制删除不提示
复制
cp hello.txt /home/bbb ：将文件复制进指定文件夹
cp -r /home/bbb /opt：将指定文件夹整个复制进另外一个文件夹
\cp -r /home/bbb /opt :强制覆盖不提示是否
移动/重命名
mv oldname newname：重命名，需要在同个目录下
mv /temp/file /targetfolder：移动文件，不同目录
查看
cat  -n 文件：查看文件并显示行号
cat  -n 文件 | more：按enter一行一行出来，按空格一页一页出来
more 文件：按页显示文本内容，空格向下翻页，enter向下翻一行，q离开，ctrol f向下滚动一屏，ctrol b返回上一屏，=输出当前行号，:f输出文件名、当前行号
less 文件：分屏查看文件内容，空格向下翻页，pgdn向下翻页，pgon向上翻页，/字串 向下搜寻字串，？字串 向上搜寻，q离开
echo $HOSTNAME：输出内容到控制台
头尾查看
head 文件：查看文件前10行，默认前10行
head -n 5 文件 ：前5行
tail 文件：尾10行
tail -n 5 文件：尾5行
tail -f 文件：实时追踪文档的所有更新
cut
cut -f 1,2 a.txt:过滤a.txt的第一、二列的信息
重定向  
写
>重定向   >>追加
ls -l > a.txt：列表的内容写入a.txt文件,并删除原本a.txt里的内容，覆盖
ls -la >> a.txt: 列表的内容追加到a.txt的末尾
cat 文件1 > 文件2：将文件1的内容覆盖到文件2
读
<:读文件，  cat < a.txt
<<：读取后面的内容      
<<- ：允许tab，美观 
‘EOF’：普通文本，
EOF、"EOF"：会识别变量$name

cat > a.txt <<- EOF
    tom=$user
EOF

#上面的写法不会再终端回显

tee a.txt << 'EOF'
hello
EOF
软链接
ln -s 原文件 软链接名：给源文件创建一个软链接    使用绝对路径
ln -s /home/myroot /root  ：如果cd myroot访问的是root的目录
查看历史指令
history
history 10：最近10个
时间日期
date：显示当前时间
date -s “2020-0101 20:20:20”：设置当前系统时间 
cal：显示本月日历
cal 2026：显示2026年所有的日历
搜索查找
find
find 搜索范围  搜索方式 文件名
 搜索方式：-name <指定文件名查找> -user<用户名> 查找指定用户名的所有文件 -size <文件大小 K M G> 指定文件大小查找
find /home -name hello.txt
find / -size +200M 大于200M的文件，-就是小于
-lh  按照单列有单位的显示
find 文件 -mtime n：n天前动过的文件，+n：大于n天，-n：n天内
locate
locate 文件
使用locate搜索文件需要在第一次运行前，使用updatedb指令创建locate数据库
grep
过滤查找
.... | grep ...    
cat /home/txt | grep yes
grep -c "字符串" file:对file内出现的字符串进行统计
压缩&解压
gzip
gzip 文件：将文件压缩成.gz的文件
gunzip 文件.gz  :解压.gz文件
zip
zip  xxx.zip 文件：将文件压缩成xxx.zip
zip -r xxx.zip 目录 ：整个目录压缩成xxx.zip
unzip 文件：解压zip文件
unzip -d 解压到的目录 zip文件：将文件解压到指定目录
zip -r home.zip /home/
unzip -d /opt/tmp /home/home.zip
tar
tar 选项 xxx.tar.gz 打包文件/目录：将文件/目录打包成.tar.gz格式
选项：-c 产生tar打包文件，-v 显示详细信息，-f 指定压缩后的文件名，-z 打包同时压缩，-x 解包.tar文件
tar -zcvf pc.tar.gz /home/pig.txt /home/cat.txt:同时压缩多个指定文件
tar -zcvf home.tar.gz /home/
tar -zxvf pc.tar.gz
tar -xvf pc.tar
tar -zxvf /home/home.tar.gz -C /opt/tmp :将文件解压到指定文件夹
